new sap.f.semantic.SemanticToggleButton(sId?, mSettings?)

Constructor for a new SemanticToggleButton.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • pressed : boolean (default: false)

In addition, all settings applicable to the base type sap.f.semantic.SemanticButton can be used as well.

Parameters:
stringsId?ID for the new control, generated automatically if no ID is given
objectmSettings?Initial settings for the new control

getPressed(): boolean

Gets current value of property pressed.

Defines the SemanticToggleButton pressed state.

The property is set to true when the control is toggled (default is false).

Default value is false.

Returns:
