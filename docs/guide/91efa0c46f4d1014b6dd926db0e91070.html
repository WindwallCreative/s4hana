Demo Kit

The OpenUI5 documentation is included in the Demo Kit. The Demo Kit also provides you with technical documentation and samples.

Demo Kit

The Demo Kit contains the following:

  • Developer Guide with the OpenUI5 documentation.

  • Explored app, which gives a detailed view of almost every control, including detailed information about the corresponding properties, aggregations, events, and methods. In addition, it also provides samples including a code view that allows you to easily copy any code snippets you may need. See Explored.
    Note

    You can also view and download the code for our tutorials from the Explored app. Just search for “Tutorial”. For more information, see Tutorials.

  • API Reference, which includes JavaScript documentation for the framework and control API. See API Reference.

  • Demo Apps, which also showcase actual samples. See Demo Apps.

  • Icons with an overview of all available icons with OpenUI5. See Icons.

View the SAPUI5 Demo Kit at https://sapui5.hana.ondemand.com/, and the OpenUI5 version at https://openui5.hana.ondemand.com/. If you want to see the Demo Kit for a specific version, simply add the version number to the URL, for example https://sapui5.hana.ondemand.com/1.42.7/ or https://openui5.hana.ondemand.com/1.42.7/
Note

Check the available versions with respective maintenance status at https://openui5.hana.ondemand.com/versionoverview.html

For the OpenUI5 version of the Demo Kit, we also provide a beta version of the upcoming release at https://openui5beta.hana.ondemand.com/ and a preview of the version that is currently in development at https://openui5nightly.hana.ondemand.com/.
